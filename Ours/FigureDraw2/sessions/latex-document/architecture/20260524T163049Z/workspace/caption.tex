\caption{\textbf{RetroDiff system architecture.} Data flows from a \textit{Query} input (blue) through the \textbf{Retrieval} group (shaded), where a \textit{Retriever} module queries a \textit{Knowledge Base} (KB) datastore (dashed arrow indicates a query/feedback edge). Retrieved context is passed to the \textbf{Generation} group: the \textit{Diffusion Decoder} conditions on it and is guided by a \textit{Prior Network}, ultimately producing a \textit{Generated Sample} output (green). Solid arrows indicate forward data flow; dashed arrows indicate retrieval queries.}

\caption{\textbf{RetroDiff system architecture.} Solid arrows indicate data flow. The \textit{Retrieval} group (blue dashed boundary) comprises a Retriever module and a Knowledge Base (KB) datastore; the \textit{Generation} group (orange dashed boundary) comprises a Diffusion Decoder and a Prior Network. An input query is passed to the Retriever, which queries the KB and forwards retrieved context to the Diffusion Decoder; the Decoder conditions on the Prior Network to produce the final generated sample.}

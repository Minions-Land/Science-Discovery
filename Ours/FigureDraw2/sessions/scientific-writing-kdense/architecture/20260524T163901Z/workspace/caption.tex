\caption{\textbf{Architecture of RetroDiff.} Solid arrows denote forward data flow; dashed arrows denote control/feedback connections. The \textit{Retrieval} group (blue border) comprises the \textit{Retriever} module and the knowledge-base (\textit{KB}) datastore, which supplies relevant context to the \textit{Diffusion Decoder}. The \textit{Generation} group (green border) contains the \textit{Diffusion Decoder} and a \textit{Prior Network} that provides a conditioning signal via a feedback path. The decoder fuses retrieved context with the learned prior to produce the final \textit{Generated Sample}.}

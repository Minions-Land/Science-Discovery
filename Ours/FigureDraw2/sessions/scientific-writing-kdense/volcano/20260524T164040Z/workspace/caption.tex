\caption{Volcano plot of differential expression for 3{,}000 genes. Each point represents one gene plotted as $\log_2$ fold change (x-axis) versus $-\log_{10}(p)$ (y-axis). Genes exceeding both thresholds ($|\log_2\text{FC}|>1$, $-\log_{10}p>2$) are coloured red (up-regulated, $n=235$) or blue (down-regulated, $n=230$); remaining genes are grey ($n=2{,}535$). Dashed lines mark the significance thresholds. The five genes with the highest $-\log_{10}(p)$ values are labelled (G\textit{i}, where $i$ is the 1-based index in the input data). Gene identifiers were not provided in the input data; labels are index-based placeholders.}

\caption{%
  Ridgeline plot of log-expression distributions across six transcriptomic clusters.
  Each ridge shows a kernel-density estimate (Gaussian kernel, Scott's bandwidth rule)
  for the cells within that cluster; ridges are stacked with mild vertical overlap to
  facilitate direct shape comparison.
  \textbf{Cluster 6 displays the highest median log-expression ($\tilde{x}\approx3.3$)
  and the widest spread}, while Cluster 1 sits lowest ($\tilde{x}\approx2.0$), indicating
  a progressive shift in expression level from cluster 1 to cluster 6.
  Color encodes cluster order (pale blue: Cluster 1 → deep blue: Cluster 6);
  the middle ridge (Cluster 3/4) is drawn with a slightly heavier stroke to anchor the visual.
  $N=300$--$500$ cells per cluster; density curves are scaled uniformly so ridge heights
  are directly comparable across clusters.
}

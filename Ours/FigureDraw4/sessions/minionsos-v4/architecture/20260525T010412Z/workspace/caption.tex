\caption{\textbf{RetroDiff system architecture.} Pipeline overview of RetroDiff, a retrieval-augmented diffusion model. A query enters the \textit{Retrieval} module, where the Retriever queries a knowledge base (KB) and returns retrieved context (solid arrows, data flow). Retrieved representations are passed to the \textit{Generation} module, where the Diffusion Decoder produces the output sample; the Prior Network conditions the decoder via iterative feedback (dashed arrows, control/feedback). No quantitative data are shown; diagram is derived entirely from \texttt{data.json}.}

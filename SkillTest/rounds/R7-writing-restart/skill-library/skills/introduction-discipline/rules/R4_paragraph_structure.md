**规则**：Introduction 应有 4–6 段 hourglass 结构。

**触发**：Intro 段落数 < 4 时。

**结构模板**：
1. **Broad context**（1 段）：领域背景 + 为什么重要
2. **Gap / problem**（1 段）：现有方法的不足 + 为什么难
3. **Our approach**（1–2 段）：方法类概述 + key insight
4. **Contribution**（1 段）：contribution bullets
5. **(Optional) Roadmap**（1 段）：论文结构导览

**注意**：CNS 论文可以压缩到 4 段；会议论文通常 5–6 段。少于 4 段说明内容不够充实。

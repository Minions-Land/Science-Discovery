**规则**：Related Work 最少 1 整页（~600–800 词）。

**触发**：Related Work 字数 < 600。

**原因**：短 Related Work 是 reviewer 常见投诉。3–4 个 substantive paragraphs 是最低要求。

**修复**：如果太短，检查是否遗漏了某个方法类。常见遗漏：
- 只写了最近的 SOTA，没写经典方法
- 只写了同领域，没写跨领域的相关技术
- 只写了方法，没写 benchmark/evaluation 相关工作

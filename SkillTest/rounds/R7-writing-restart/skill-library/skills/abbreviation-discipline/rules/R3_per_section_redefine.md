**规则**：Abstract 和 Intro 各自独立定义简写。

**要求**：因为有些读者只看 Abstract 或只看某个 section，所以：
- Abstract 里第一次出现的缩写要定义
- Introduction 里第一次出现的缩写也要定义（即使 Abstract 已定义过）
- Related Work 之后的 section 可以假设读者已经读过前面的定义

**注意**：这不意味着每个 section 都要重新定义——只是 Abstract 和 Intro 各自独立。

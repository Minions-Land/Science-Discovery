**规则**：报告不同超参数下的结果。

**要求**：
- 关键超参数（learning rate, hidden dim, number of layers, temperature 等）做 sensitivity analysis
- 通常用一个小 figure 或 table 展示
- 说明选择当前值的理由

**注意**：不需要 grid search 所有超参——只需要展示对结果影响最大的 2–3 个。

# Abstract: Neural Mechanisms of Memory Consolidation

Memory consolidation is a critical process in neuroscience that transforms short-term memories into stable, long-term memories. The hippocampus (HC) is essential for this process, particularly during sleep. Recent studies suggest that rapid eye movement sleep (REMS) plays a key role in memory consolidation. However, the specific molecular mechanisms remain unclear. In this study, we investigated the role of brain-derived neurotrophic factor (BDNF) in memory consolidation during REMS. We used transgenic mice with conditional knockout of BDNF in the HC to examine memory performance. Our results show that BDNF in the HC is critical for memory consolidation during REMS. Specifically, loss of BDNF in the HC impaired both contextual and spatial memory tasks. Furthermore, we found that BDNF levels increase during REMS in the HC. These findings suggest that BDNF is a key molecular mechanism underlying memory consolidation during REMS. Our work provides new insights into the molecular basis of memory consolidation and may have implications for understanding memory deficits in neurological disorders.

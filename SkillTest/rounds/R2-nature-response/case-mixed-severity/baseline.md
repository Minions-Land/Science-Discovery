# Response to Reviewers

Dear Editor,

We appreciate the reviewers' careful assessment and have revised the manuscript to strengthen the evidence base while keeping the claims aligned with the data.

**Reviewer 1, Comment 1:** The reviewer asked for replication in a second mouse strain with a larger cohort and neuron-subtype stratification. We agree. We will add a replication experiment in a second strain with n=12 per group and report editing efficiency separately for excitatory and inhibitory neurons. These data will be added to the Results, Fig. 2, and a new supplementary table with full cohort information.

**Reviewer 1, Comment 2:** The reviewer noted that the truncated y-axis in Fig. 2c may exaggerate regional differences. We partially agree. We will re-plot the main Fig. 2c using a 0-100% y-axis so the absolute effect size is clear. Because the original axis was chosen to show variance within the edited range and follows FDA-style graphical guidance for bounded percentage endpoints, we will also explain this in the figure caption and move the truncated version, if retained, to the supplement.

**Reviewer 1, Comment 3:** The reviewer suggested removing the clinical relevance claim because the cholesterol reduction is modest relative to statins. We respectfully disagree with removing the claim entirely. The effect size is not intended to be presented as magnitude-equivalent to systemic statin therapy. Its relevance is that a CNS-targeted PCSK9 intervention produced a measurable cholesterol phenotype through a mechanistically distinct route. We will soften the Discussion to avoid overstating translational readiness, clarify that the result is mechanistic rather than directly competitive with statins, and cite the revised text on page 12, lines 310-324.

**Reviewer 2, Comment 1:** The reviewer requested chemical characterisation of compound 14B. We agree. We will provide 1H-NMR, mass spectrometry, and HPLC purity data in the supplement, with methods added to page 6 and the spectra and chromatogram placed in Supplementary Fig. S4.

**Reviewer 2, Comment 2:** The reviewer asked for a peptide-free control to support the BBB-crossing mechanism. We agree. We will add a peptide-free LNP control and report brain editing efficiency and biodistribution alongside the targeting-peptide formulation. The comparison will be added to Fig. 2 and Supplementary Fig. S5. We will also revise the mechanism language so that the BBB-crossing claim depends directly on this control.

## Revision notes

- Accepted the major experimental requests for strain replication, chemical characterisation, and peptide-free control because each strengthens a central evidentiary claim.
- Partially accepted the Fig. 2c axis request by changing the main display while preserving a caption explanation for the original scaling rationale.
- Declined full removal of the clinical relevance discussion, but narrowed the claim to mechanistic novelty of CNS-only PCSK9 modulation.
- Added concrete destinations for evidence in figures, supplement, methods, and discussion text.

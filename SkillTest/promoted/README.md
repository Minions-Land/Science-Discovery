# Promoted Skills

Skills tested in SkillTest rounds and judged worth keeping for future use.
**This is a research library, not a production library.** Nothing here is
loaded by MinionsOS automatically. Skills here are organised by topic so
you can browse "what we know works" by domain.

## Structure

```
promoted/
├── README.md                  # this file
├── academic-writing/          # prose polishing, abstract structure, register discipline
├── academic-figures/          # figure rcParams, palette, layout discipline
├── review-quality/            # criticism specificity, evidence boundary
└── submission-mechanics/      # data availability, citation discipline, no-fabrication
```

Each category has its own README explaining what was tested, which rounds
the evidence comes from, and the specific rules / patterns that were
extracted. The original third-party skill files are *referenced by path*
rather than duplicated when they are in `/Users/mjm/Skill/`; SkillTest-
authored drafts and updates are placed inside the category.

## How to read a category

For each category README:

1. **Skills inside**: third-party skills (with original path) plus
   SkillTest-authored drafts.
2. **Rules / patterns extracted**: the load-bearing transferable insights,
   each cited to a round + case.
3. **Recommendation**: import-strongly / fork-narrowly per skill, with
   justification.
4. **Open questions**: what the next round of testing should sharpen.
